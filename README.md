# LegalFlow

## Overview
LegalFlow is a CLI-based legal case management system designed for law firms. It helps manage clients, lawyers, and legal cases in one organized system. It also includes a smart priority feature that identifies the highest-priority case based on case importance, type, status, and hearing date.

## Users
- Client
- Lawyer
- Admin

## Features

### Client
- Add new case
- View my cases

### Lawyer
- View assigned cases
- Update case status
- Show highest-priority case
- Check upcoming hearings

### Admin
- Add client
- Add lawyer
- View all cases
- View all clients

## Smart Feature
The system calculates a priority score for each case and helps the lawyer identify which case should be handled first.
